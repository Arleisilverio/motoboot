# Project Skills & Specialist Agents

Este diretório contém "Habilidades" (Skills) e manuais de "Agentes Especialistas" para este projeto.

## Como usar este diretório?

Se você é um agente de IA (como Cursor, Windsurf, Claude ou Copilot), este diretório é sua fonte de verdade para tarefas especializadas.

### Estrutura Sugerida
Cada subpasta deve representar uma especialidade:
- `skills/n8n-expert/`: Instruções sobre como criar workflows específicos para este projeto.
- `skills/supabase-architect/`: Padrões de banco de dados e políticas RLS.
- `skills/premium-design/`: Tokens de design e diretrizes visuais do Motoboot.

### O Arquivo SKILL.md
Cada pasta de habilidade deve conter um arquivo `SKILL.md` com:
1. **Contexto**: Quando usar esta habilidade.
2. **Regras**: O que fazer e o que não fazer.
3. **Exemplos**: Trechos de código ou comandos que funcionam neste ambiente.

## Vantagem desta Técnica
Ao manter as habilidades aqui, garantimos que o projeto seja **Agent-Friendly**, permitindo que diferentes IAs colaborem mantendo o mesmo padrão de qualidade e arquitetura.
